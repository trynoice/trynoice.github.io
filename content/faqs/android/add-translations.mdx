title: Where can I add translations for the Android app?
---

import { Image } from "@chakra-ui/react";

You can help us in translating Noice Android app on the [Weblate
Project](https://hosted.weblate.org/engage/noice/). There you can find an
existing translation for a language. You can also add a new language if it
doesn't already exist.
